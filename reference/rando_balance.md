# rando_balance demonstration dataset

A synthetic dataset used in examples. The dataset contains two
randomisation result variables (`rando_res` and `rando_res2`) and two
stratification variables (`strat1` and `strat2`).

## Usage

``` r
rando_balance
```

## Format

### `rando_balance`

A data frame with 100 rows and 3 columns:

- strat1, strat2:

  Binary stratification variables

- rando_res:

  Balanced randomisation result

- rando_res2:

  Unbalanced randomisation result
